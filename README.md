# 电子产品零售数据可视化系统 python100

## 项目概述

本项目是基于Python的电子产品零售数据可视化系统，致力于通过数据爬取、清洗、分析与可视化的方式，全面揭示电商平台中电子产品销售信息的内在规律与市场趋势。

## 技术栈

- **后端**: Python
- **Web框架**: Flask
- **数据分析**: Pandas
- **数据可视化**: Echarts, wordcloud, matplotlib
- **数据库**: Mysql

## 功能模块

### 数据处理

- 数据爬取：自动获取电商平台销售数据
- 数据清洗：对获取的数据进行格式化与清洗，以适应分析需求

### 数据分析

- 店铺分析表：比较各店铺的销售情况
- 评分分析表：用户评分分布及好评率趋势
- 区间分析占比表：各类指标的区间分布比例

### 数据可视化

- 品牌词云图：统计品牌热词
- 店铺词云图：展示店铺热词
- 综合词云图：综合展示关键信息

### 用户数据管理

- 用户数据浏览：查询平台用户信息
- 用户数据维护：对用户信息进行管理，支持系统使用和权限控制

## 系统角色

- 系统管理员：负责系统的维护与数据管理
- 普通用户：浏览与分析数据

## 运行环境

- Python环境
- Flask框架
- Mysql数据库

## 部署步骤

1. 安装Python环境
2. 配置Mysql数据库
3. 安装Flask框架
4. 部署后端服务
5. 部署前端页面

```
/
```
